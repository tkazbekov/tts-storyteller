# Documentation

TTS Storyteller is a local multi-backend TTS API/CLI for multi-voice story generation.

Start with the root `README.md` for setup and day-to-day commands.

## Files

- `ARCHITECTURE.md` - system architecture and data flow
- `BACKENDS.md` - Qwen3-TTS and VibeVoice backend notes
- `SWAGGER.md` - API usage examples and Swagger UI notes
- `web-app.md` - notes for the companion web app
- `openapi.json` - exported OpenAPI schema
- `story-template.schema.json` - JSON schema for story templates
- `features/` - implementation notes for file upload, voice cloning, and VibeVoice support

## Local API docs

Run the API and open:

```text
http://localhost:8000/docs
```
